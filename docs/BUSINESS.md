# Бизнес-цель

Django-бэкенд — «мозг» платформы BeatVault по продаже MTProto-прокси ключей через Telegram-бота.

## Что продаём

Пользователь получает временную ссылку на MTProto-прокси (`tg://proxy?...`). Ссылка даёт быстрый доступ к Telegram в странах, где мессенджер замедлен или заблокирован. Одна ссылка работает на 3 устройствах одновременно.

## Монетизация

Два способа оплаты:

| Провайдер | Цена | Механизм |
|-----------|------|----------|
| ЮKassa (Yandex) | 79 ₽/мес | Telegram Payments API |
| Telegram Stars | 60 ★/мес | Встроенная валюта Telegram |

При покупке: если у пользователя есть активный ключ — продлевается на 30 дней. Если нет — генерируется новый.

## Бесплатная ссылка

Каждый новый пользователь получает бесплатный пробный период:

| Условие | Период |
|---------|--------|
| Обычный пользователь | 30 дней |
| Пришёл по реферальной ссылке | 14 дней |
| Лимит бесплатных выдач (`FIRST_MONTH_LIMIT`) исчерпан | 7 дней |

Бесплатный период выдаётся один раз.

## Реферальная программа

Пользователь делится ссылкой `https://t.me/bot/?start=<telegram_id>`. После того как 5 приглашённых активируют свой бесплатный период, пригласивший может забрать бесплатную 14-дневную ссылку. Лимит — `REFERRAL_LINKS_LIMIT` активаций.

## Жизненный цикл ключа

1. **Создание** — бэкенд выбирает наименее нагруженный VDS, отправляет POST на FastAPI-сервис, получает секрет и домен, формирует `tg://proxy` ссылку.
2. **Использование** — пользователь нажимает ссылку, Telegram подключается к прокси.
3. **Перевыпуск** — пользователь запрашивает через бота. Генерируется новый секрет, старый удаляется. Кулдаун — 5 минут.
4. **Истечение** — Celery Beat ежедневно удаляет протухшие ключи с VDS, деактивирует записи в БД, уведомляет пользователей.

## Уведомления

| Время (UTC) | Что происходит |
|-------------|---------------|
| 15:00 | Предупреждение за 1 день до истечения |
| 08:00 | Предупреждение за 1 час до истечения |
| 09:00 | Удаление истёкших ключей с VDS |
